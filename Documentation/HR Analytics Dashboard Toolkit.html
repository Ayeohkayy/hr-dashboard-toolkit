First Time Setup


Follow the steps below to connect your Excel data and open the HR Analytics Dashboard for the first time.


Step 1: Save the Toolkit Folder


After downloading the toolkit ZIP file:

  • Extract the folder to your computer.

  • Save the folder somewhere easy to access.


Important:

You can rename the folder and Excel files if you want.
However, do not change the column headers or Excel table names inside the files, as the dashboard depends on these.


Step 2: Add Your HR Data


  • Open the Excel tracker files included in the toolkit:

Employee_Data.xlsx
Recruiting_Data.xlsx

These spreadsheets are where your HR data will be stored and maintained.


  • Enter or paste your employee and recruiting information into the appropriate columns.


You may safely:

• Add new rows of data
• Update employee information
• Add recruiting activity and candidate records

To ensure the dashboard continues to work correctly, do not:

• Rename column headers
• Rename Excel tables
• Delete required columns


  • Once your data has been entered, save and close the Excel files.


This Excel file will be the file you continuously use to manage and update your HR data going forward.


Step 3: Download Power BI


The HR Analytics Dashboard is built using Microsoft Power BI Desktop, a free data visualization tool used to create interactive dashboards.

If you do not already have Power BI installed on your computer, follow the steps below to download and install it.

Download Power BI Desktop

  • 1. Open your web browser.

  • 2. Go to the official Microsoft Power BI download page: https://powerbi.microsoft.com/desktop/

  • 3. Click Download Free.

  • 4. This will redirect you to the Microsoft Store page for Power BI Desktop.

  • 5. Click Install.

  • 6. Once the installation is complete, open Power BI Desktop from your computer.



Step 4: Open the Dashboard Template


  • Open the Power BI template file included in the toolkit:

HRDashboard.pbit

When the template opens, Power BI will prompt you to provide the file location of your Excel trackers. As shown in the figure below.


Step 5: Connect Your Excel Files


  • Locate your Excel files on your computer:

Employee_Data.xlsx
Recruiting_Data.xlsx

  • Right-click each file and select Copy as path.

  • Paste the file path into the Power BI prompt when asked for:

Select Employee Data File
Select Recruiting Data File

  • Once both file paths are entered, click Load.

Power BI will now connect to your Excel data and automatically build your HR analytics dashboard.


Important

Sometimes Windows adds quotation marks around the file path when copying it. If this happens, remove the quotation marks before clicking Load.

Correct format

C:\Company\HR Analytics\Employee_Data.xlsx

Incorrect format

"C:\Company\HR Analytics\Employee_Data.xlsx"


Step 6: Explore the Dashboard


After loading completes, your dashboard will appear.

You can now:

  • View workforce analytics

  • Analyze recruiting activity

  • Filter charts and visuals

  • Explore your HR insights


  • To reset all filters, click the Reset Dashboard button.



Note: When using Power BI Desktop, hold Ctrl + Click when pressing dashboard buttons.


Step 7: Refresh Data


Whenever you update your Excel trackers:

  • Save the Excel files

  • Open the Power BI dashboard

  • Click Refresh


Your charts and metrics will update automatically.

Using Excel Trackers


Important Rules


You may safely:

• Add new rows of data
• Update employee information
• Add recruiting activity and candidate records

To ensure the dashboard continues to work correctly, do not:

• Rename column headers
• Rename Excel tables
• Delete required columns


The Excel tracker files included in this toolkit are used to store your HR data. The Power BI dashboard reads these files to generate analytics and visualizations.

Follow the examples below when entering or updating your data.

Using Dropdown Menus



Several columns in the Excel trackers include dropdown menus to help standardize your data.


To use a dropdown menu:

  • 1. Click the cell in the column.

  • 2. Click the dropdown arrow on the right side of the cell. (Shown in figure above)

  • 3. Select the appropriate value from the list.


Using dropdown menus helps ensure your data remains consistent and allows the dashboard to group and analyze information correctly.

Examples of dropdown fields may include:

• Department
• Employment Status
• Performance Rating
• Gender


Updating Dropdown List Values



Dropdown menu options are controlled by the lookup tables in the Excel file.

If your organization uses different values, you can update these lists.

To modify a dropdown list:

  • 1. Locate the lookup table in the spreadsheet.

  • 2. Add or edit the values in the list.

  • 3. Save the Excel file.

The dropdown menus in the tracker will automatically update to reflect the changes.


Calculated Columns



Some columns in the tracker are automatically calculated using Excel formulas. These columns are in red font with a grey background.


Important:

Do not enter or modify data in these columns. The formulas will calculate the values automatically.


Onboarding Tracker



Each row in the Onboarding Tracker corresponds to an employee listed in the “Employees” sheet.

The connection between the two sheets is made using the Employee ID column.

When you enter or select an Employee ID, the tracker automatically retrieves related employee information from the Employees sheet. (Shown in Figure above)

This ensures that onboarding records always match the correct employee.


Updating Onboarding Status


The primary field you will update in this sheet is the Onboarding Status column.

Examples of onboarding status values may include:

• Not Started
• In Progress
• Completed
• Delayed

Updating this field allows the dashboard to track onboarding progress and identify employees who may still have outstanding onboarding tasks.


Best Practices


To keep the onboarding tracker working properly:

• Always select or enter a valid Employee ID
• Do not modify the grey formula columns
• Add new rows only at the bottom of the table
• Ensure the employee exists in the Employees sheet before adding onboarding records



Using and Navigating the Dashboard


The HR Analytics Dashboard is interactive and allows you to explore your workforce and recruiting data in multiple ways. You can filter information, navigate between dashboard pages, and interact with charts to explore trends and insights.


Create the Best Viewing Experience




When opening the dashboard in Power BI Desktop, several editing panels may appear on the right side of the screen.

For the best viewing experience, collapse these panels.

This will expand the dashboard canvas and allow you to focus on the visuals without the editing interface.

Using Dashboard Filters



The dashboard includes dropdown filters at the top of the page to help you focus on specific groups of data.

Common filters include:

• Department
• Work Location

The dashboard visuals will automatically update based on your selection.


Navigating Dashboard Pages


The dashboard contains multiple pages designed for different HR insights.

Examples include:

• Overview
• Payroll
• Recruiting
• FAQs

To navigate between pages, click the page buttons at the top of the dashboard.

Important

When using Power BI Desktop, you must hold Ctrl + Click when selecting navigation buttons.


Interacting With Dashboard Visuals



Charts and visuals on the dashboard are connected. When you click on a data point in a visual, the rest of the dashboard will automatically filter based on that selection.

Example:

If you click the “Production” bar in the Department Headcount chart, all other visuals on the page will update to display metrics only for the Production department.

To clear the selection, simply click the visual again or reset the dashboard filters.


Refreshing Data and Syncing Your Data in Power BI


The HR Analytics Dashboard reads data directly from the Excel tracker files included in this toolkit. When you update the Excel files, you will need to refresh the dashboard so Power BI can reload the latest information.

Refreshing ensures that all charts, metrics, and visuals reflect your most current HR data.


How to Refresh the Dashboard




To refresh your dashboard:

  • 1. Open your Power BI dashboard file.

  • 2. Locate the Refresh button in the top toolbar.

  • 3. Click Refresh.

Power BI will reconnect to the Excel files and update all visuals in the report. This process typically takes only a few seconds depending on the size of your dataset.


When Should You Refresh?


You should refresh the dashboard whenever changes are made to your Excel trackers.

Common examples include:

• Adding new employees
• Updating employee information
• Recording employee terminations
• Adding new recruiting candidates
• Updating recruiting stages or job statuses

Refreshing the dashboard ensures your HR analytics stay accurate and up to date.


Best Practices


  • Always save your Excel files before refreshing Power BI

  • Keep your Excel trackers in the same folder location

  • Avoid renaming Excel table names or column headers

  • Refresh the dashboard after major updates


Troubleshooting


Problem

Possible Cause

Solution

Dashboard does not update after refreshing

Excel file changes were not saved

Save the Excel tracker files and click Refresh again in Power BI

Power BI cannot find the Excel file

File was moved, renamed, or folder location changed

Reopen the Power BI template and provide the correct file path for the Excel tracker

Dashboard shows blank visuals

Filters may be applied or data may be missing

Clear filters using the Reset Dashboard button or review your Excel data

Some charts appear incorrect

Data entry inconsistency in Excel

Verify dropdown selections and confirm values follow the lookup list

Refresh fails or shows an error

Power BI temporarily lost connection to the Excel file

Close Power BI Desktop and reopen the dashboard, then refresh again

New employees or candidates do not appear

Data was added but dashboard not refreshed

Save the Excel file and click Refresh in Power BI

Dropdown values not appearing in Excel

Lookup list may not contain the value

Add the value to the Lookup Table and save the Excel file

Dashboard navigation buttons do not work

Power BI Desktop requires a special click command

Hold Ctrl + Click when selecting navigation buttons

Dashboard view feels too small

Power BI editing panels are open

Collapse the Filters, Build, and Data panels on the right side

Data appears outdated

Dashboard has not been refreshed recently

Click Refresh to sync the latest Excel data
