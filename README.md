# HR Dashboard Toolkit

## Overview
This project is a reusable HR analytics toolkit built with Excel and Power BI.

It is designed to help HR teams and small businesses track workforce and recruiting data using structured Excel templates, then visualize that information in Power BI through an interactive dashboard.

The toolkit includes:
- an employee data template
- an onboarding tracker
- a recruiting tracker
- an open positions tracker
- a Power BI dashboard connected to those Excel files

## Files Included

### Power BI
- `PowerBI/HR_Dashboard.pbix`
  - Main dashboard file
  - Connects to the Excel trackers
  - Used for reporting and analytics

### Excel Templates
- `Excel_Templates/Template.Employee.Data.xlsx`
  - Employees
  - Onboarding Tracker
  - Lookups
  - Instructions
  - FAQs

- `Excel_Templates/Template.Recruiting.Data.final.xlsx`
  - Candidate Tracker
  - Open Positions
  - Lookups
  - Instructions
  - FAQs

## Purpose
This toolkit was created to provide a practical HR reporting solution that combines easy Excel data entry with dynamic Power BI dashboards.

It can be used for:
- headcount tracking
- recruiting pipeline reporting
- onboarding visibility
- open position monitoring
- executive HR dashboarding

## How It Works
1. Users enter or update HR data in the Excel templates.
2. Power BI connects to the Excel files.
3. The dashboard refreshes to reflect updated workforce and recruiting data.
4. HR teams can use the dashboard for reporting and decision-making.

## Setup Instructions
1. Download the Excel template files and Power BI file.
2. Store the Excel files in a local folder.
3. Open the `.pbix` file in Power BI Desktop.
4. If needed, update the file path in Power BI so the dashboard points to your local Excel files.
5. Refresh the data.

## Notes
This project is intended as a reusable HR analytics template and portfolio project.

Sensitive or real employee data should never be uploaded to GitHub. Only template or sample files should be included.
